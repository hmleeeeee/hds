						<script>					
							$('.easy-pie-chart').easyPieChart({
								barColor: '#999',
								trackColor: '#e7e7e7',
								scaleColor: false,
								lineCap: 'round',
								lineWidth: 5,
								size: 60,
								animate: 1000,
								onStep: function (from, to, percent) {
									$(this.el).find('.percent').text(Math.round(percent));
								},
							});
						</script>

Chart.js

  • https://www.chartjs.org/docs/2.9.3/charts/
  • Chart.js 3.x 버전부터 IE를 지원하지 않아 2.9.3 버전 사용
  • canvas 기반이므로 별도의 CSS 파일이 존재하지 않으며 스크립트만으로 스타일 제어
  • 옵션1 : 12
  • 옵션2 : 19
  • 옵션3 : 3
  • 옵션4 : 5
  • 옵션5 : 2
 
							<!-- 사이즈 지정시 IE 호환성 이슈 발생하지 않도록 감싸는 컨테이너에 크기지정 사용 권장 -->
							<div style="width: 100%; height: 300px;">
								<canvas id="myChart1">
									<!-- [웹 접근성] canvas 내부에 마크업으로 대체콘텐츠 마크업 필수 -->
									<div class="sr-only">대체콘텐츠</div>
								</canvas>
							</div>

Chart.js-plugin

  • https://v0_7_0--chartjs-plugin-datalabels.netlify.app
  • Chart.js 데이터 값 노출 옵션 사용하기 위한 추가 플러그인
  • Chart.js 2.9.3 버전의 호환성으로 chartjs-plugin-datalabels v0.7.0 사용
  • 옵션1 : 17.7
  • 옵션2 : 11.5
  • 옵션3 : 63.6
  • 옵션4 : 7.2
 
							<script src="../../asset/plugins/chart/chartjs-plugin-datalabels.min.js"></script>
							<script>
								...
									option {
										// 반응형 조건 예시
										legend: {
											position: (window.innerWidth >= 767) ? 'left' : 'top', // 삼항연산자 좌측 Desktop&Tablet , 우측 Mobile
											align: (window.innerWidth >= 767) ? 'left' : 'center',
										}

										// 플러그인 옵션 예시
										plugins: {
											datalabels: {
												color: '#ffffff', // 컬러 지정
												anchor: 'center', // 차트 내부 데이터 위치 지정
												align: 'end', // 데이터 위치 지정
												formatter: function (value) {
													return value + '%';
												}, // 데이터 + 수식값 지정
											}
										}
									}
								...
							</script>
